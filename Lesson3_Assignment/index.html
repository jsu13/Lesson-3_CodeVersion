Highlight #1 What is version control?

"Version control acts as a single repository for all of a persons, or group of peoples files. Team members can view files, edit them, and then re-commit them back into the repository.Version control acts as a simple way to collaborate with other people durring projects. Version control can act as a way to back up your information in case you or a team member loses something."

Highlight #2 What is Git

Git is and open source distributed version system that is designed to handle everything from small, to large projects with speed and effiency.Git is useful because it allows for safe data work between groups of people which helps effiency for getting projects done.

Highlight #3 Git Commands

The three basic steps for Git are to add, commit, and then push. Before stating the command the word "git" needs to be used. Some of the basic comands include init, add --all, commit -am 'inital commit', log, and status. The command terminal can be found by going to tools on codio, where all of the commands can be entered in regard to your files linked with github.
